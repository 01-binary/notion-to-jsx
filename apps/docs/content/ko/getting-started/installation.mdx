# 설치

## 요구사항

- **Node.js** 18 이상
- **React** 19+
- **Next.js** 14+ (권장, 서버 컴포넌트 활용)

## 패키지 설치

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
    ```bash copy
    pnpm add notion-to-utils notion-to-jsx
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    npm install notion-to-utils notion-to-jsx
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash copy
    yarn add notion-to-utils notion-to-jsx
    ```
  </Tabs.Tab>
</Tabs>

`notion-to-utils`는 Notion API에서 데이터를 가져오고, `notion-to-jsx`는 블록을 React 컴포넌트로 렌더링합니다.

## CSS 설정

루트 레이아웃에서 스타일시트를 import하세요:

```tsx copy filename="app/layout.tsx"
import 'notion-to-jsx/dist/index.css';
import 'prismjs/themes/prism-tomorrow.css'; // 코드 구문 강조 (선택)
```

> [!IMPORTANT]
> import 순서가 중요합니다: `notion-to-jsx/dist/index.css`를 Prism.js 테마와 커스텀 스타일보다 **먼저** import해야 오버라이드가 올바르게 동작합니다.

## Peer 의존성
