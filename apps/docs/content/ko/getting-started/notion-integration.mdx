# Notion 연동

Notion에서 데이터를 가져오려면 **내부 통합 토큰**과 **연결된 데이터베이스**가 필요합니다.

<Steps>

### 통합 생성

1. [My Integrations](https://www.notion.so/my-integrations)에 접속
2. **새 통합** 클릭
3. 이름 설정 (예: `my-blog`)
4. 워크스페이스 선택
5. **Internal Integration Secret** 복사 — 이것이 `NOTION_TOKEN`입니다

### 데이터베이스 연결

1. 사용할 Notion 데이터베이스 열기
2. **···** 메뉴 → **연결** 클릭
3. 생성한 통합을 검색하여 연결

### 데이터 소스 ID 가져오기

데이터베이스의 **세팅** 메뉴 → **데이터 소스 관리** → **데이터 소스 ID 복사**를 클릭하세요.

이것이 `NOTION_DATABASE_ID`입니다.

![데이터 소스 ID 가져오기](/images/get-datasource-id.gif)

### 환경 변수 설정

```bash copy filename=".env.local"
NOTION_TOKEN=ntn_xxxxxxxxxxxxxxxxxxxx
NOTION_DATABASE_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

</Steps>

> [!WARNING]
> 통합에 최소 **콘텐츠 읽기** 권한이 필요합니다. 통합 설정 페이지에서 구성하세요.
