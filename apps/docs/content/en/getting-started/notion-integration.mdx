# Notion Integration

To fetch data from Notion, you need an **Internal Integration Token** and a **connected database**.

> [!TIP]
> To get started quickly, duplicate the [blog database template](https://likeable-eggnog-5b5.notion.site/3019c6bf2b1780d19126df1136126616?v=3019c6bf2b178094a9d1000c960bac28&pvs=73).

<Steps>

### Create an integration

1. Go to [My Integrations](https://www.notion.so/my-integrations)
2. Click **New integration**
3. Name it (e.g. `my-blog`)
4. Select the workspace
5. Copy the **Internal Integration Secret** — this is your `NOTION_TOKEN`

### Connect a database

1. Open the Notion database you want to use
2. Click the **···** menu → **Connections**
3. Search for your integration and connect it

### Get the data source ID

Click the **Setting** menu on your database → **Manage data sources** → **Copy data source ID**.

This is your `NOTION_DATABASE_ID`.

![Get data source ID](/images/get-datasource-id.gif)

### Set environment variables

```bash copy filename=".env.local"
NOTION_TOKEN=ntn_xxxxxxxxxxxxxxxxxxxx
NOTION_DATABASE_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

</Steps>

## Database properties

The [blog template](https://likeable-eggnog-5b5.notion.site/3019c6bf2b1780d19126df1136126616?v=3019c6bf2b178094a9d1000c960bac28&pvs=73) uses the following properties:

| Property | Type | Description |
|----------|------|-------------|
| `Name` | title | Post title |
| `Desc` | rich_text | Post description |
| `Slug` | rich_text | URL path (e.g. `my-first-post`) |
| `Category` | select | Category |
| `Date` | date | Published date |
| `isPublished` | checkbox | Publish status |

> [!WARNING]
> The integration needs **Read content** capability at minimum. Configure this in the integration settings page.
